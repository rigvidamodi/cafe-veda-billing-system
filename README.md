# ☕ Café Vèda

A simple Restaurant Billing System built using Python.

---

## 📖 About

Café Vèda is a beginner-friendly Python project that simulates a restaurant billing system.

The program allows customers to:

- View the menu
- Order food items
- Calculate the bill
- Display the final receipt
